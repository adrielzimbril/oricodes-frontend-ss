# Oricodes

Static marketing website for Oricodes, published on GitHub Pages.

[Version française](./README.md)

## Overview

This repository contains a static landing page built with HTML, CSS, and JavaScript, using Bootstrap, GSAP, and a few front-end plugins.

The site is designed for simple deployment with no build step and no back-end. All files are served directly from the repository.

## Live Demo

- Live site: `https://adrielzimbril.github.io/oricodes-frontend-ss/`
- GitHub repository: `https://github.com/adrielzimbril/oricodes-frontend-ss`

## Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- GSAP
- jQuery

## Project Structure

- `index.html`: homepage
- `contact.html`: public contact page without exposed third-party embed
- `css/`: compiled CSS
- `js/`: front-end scripts
- `image/`: images, logos, and favicons
- `plugins/`: bundled front-end dependencies
- `scss/`: SCSS sources

## Deployment

The site is deployed through GitHub Pages from the `master` branch.

There is no required build step. After making changes, pushing the static files to the repository is enough.

## Notes

- The Typeform widget was removed from the contact page to avoid exposing a third-party identifier in a public repository.
- The repository was cleaned up for public release.
