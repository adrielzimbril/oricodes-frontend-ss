# Oricodes

Site vitrine statique d'Oricodes, prêt à être publié sur GitHub Pages.

## Publication

Ce dépôt ne dépend d'aucun build. Les fichiers HTML, CSS, JS et assets sont servis tels quels.

## Pages

- `index.html` : page d'accueil
- `contact.html` : page de contact sans intégration tierce exposée côté client
