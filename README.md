# Oricodes

Site vitrine statique d'Oricodes, publié sur GitHub Pages.

[English version](./README.en.md)

## Aperçu

Ce dépôt contient une landing page statique construite en HTML, CSS et JavaScript, avec Bootstrap, GSAP et quelques plugins front-end.

Le site est pensé pour une publication simple, sans build ni back-end. Tous les fichiers sont servis tels quels depuis le dépôt.

## Démo

- Site en ligne : `https://adrielzimbril.github.io/oricodes-frontend-ss/`
- Dépôt GitHub : `https://github.com/adrielzimbril/oricodes-frontend-ss`

## Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- GSAP
- jQuery

## Structure

- `index.html` : page d'accueil
- `contact.html` : page de contact publique sans intégration tierce exposée
- `css/` : CSS compilé
- `js/` : scripts front-end
- `image/` : images, logos et favicons
- `plugins/` : dépendances front-end embarquées
- `scss/` : sources SCSS

## Déploiement

Le site est déployé sur GitHub Pages depuis la branche `master`.

Il n'y a pas d'étape de build obligatoire. Après modification, il suffit de pousser les fichiers statiques vers le dépôt.

## Notes

- Le widget Typeform de la page contact a été retiré pour éviter d'exposer un identifiant tiers dans un dépôt public.
- Le dépôt a été nettoyé pour une publication publique plus propre.
